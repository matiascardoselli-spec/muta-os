# Metodología FV

Tres skills para la capa de estrategia creativa: entender el mercado, destilar la psicología del cliente y generar ángulos para comunicar. Basado en la metodología de **Felipe Vergara** (7 Maletas de Cualquier Compra y matriz de diversificación creativa), implementada como skills por Muta Digital.

## Skills

| Skill | Qué hace | Cuándo se activa |
|---|---|---|
| **7-maletas** | Investigación de mercado automatizada: tu negocio, competidores, reviews, redes y anuncios → reporte de las 7 Maletas | "investigá el mercado", "7 maletas", "analizá mi negocio", "qué dicen mis clientes" |
| **detective-angulos** | Psicología del consumidor en 9 secciones → insights, ángulos y guía creativa para video | "qué ángulos usamos", "por qué no convierte", "qué le duele al cliente", "brief creativo" |
| **matriz-diversificacion** | Matriz de deseos × niveles de conciencia → 30 hooks con su ángulo estratégico | "matriz de diversificación", "diversificá los ángulos", "hooks por nivel de conciencia" |

## Cómo se usan

**Cada skill funciona sola.** Podés correr solo la matriz si ya conocés al cliente, o solo el detective si ya tenés investigación hecha. Ninguna exige haber corrido las anteriores.

**También se encadenan**, en este orden:

```
7-maletas (data del mercado)
      ↓
detective-angulos (psicología + ángulos)
      ↓
matriz-diversificacion (30 hooks por deseo y nivel de conciencia)
```

Cuando se encadenan, cada skill toma lo previo como contexto que enriquece, nunca como límite: si ya hay ángulos o hooks de un paso anterior, la siguiente genera los suyos distintos, y entrega igual su cantidad completa.

## Iteración mes a mes

La matriz está pensada para usarse repetidamente sobre el mismo producto. Pasale como input qué deseos y ángulos ya se trabajaron: no repite la combinación deseo + ángulo, pero sí puede volver sobre un mismo deseo desde una entrada nueva. Los deseos de un mercado son finitos y se agotan; los ángulos para expresarlos no.

## Atribución

Metodología original de Felipe Vergara: las 7 Maletas de Cualquier Compra y la matriz de diversificación creativa (deseos × niveles de conciencia de Eugene Schwartz). Estas skills son una implementación propia de Muta Digital de esa metodología, no un producto oficial ni afiliado.
