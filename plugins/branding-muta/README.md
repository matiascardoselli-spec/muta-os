# branding-muta

Skill del proceso de branding de Muta Digital:

- **informe-de-mercado** — Investigación de la categoría en la que juega un cliente o prospecto: mercado, casos homólogos internacionales, competidores locales, ángulos y lectura del terreno. Tono de analista, con evidencia y fuentes, nunca prescriptivo. Doble uso: base de entendimiento para el equipo de diseño al inicio de un proyecto de identidad, y pieza que se entrega al cliente junto con el manual de marca.

Entrega HTML con estética Muta y pasa siempre por revisión humana antes de llegar al cliente.

Origen: proceso ejecutado y validado manualmente (caso "Antes de Cebar", julio 2026) antes de documentarse como skill.
