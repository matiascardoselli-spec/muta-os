# branding-muta

Skills del proceso de branding de Muta Digital. Dos piezas que se complementan pero funcionan solas:

- **informe-de-mercado** — Investigación de la categoría en la que juega un cliente o prospecto: mercado, casos homólogos internacionales, competidores locales, ángulos y lectura del terreno. Tono de analista, con evidencia y fuentes, nunca prescriptivo. Doble uso: base de entendimiento para el equipo de diseño al inicio de un proyecto de identidad, y pieza que se entrega al cliente junto con el manual de marca.
- **hoja-de-ruta** — El destilado limpio de una reunión estratégica con el cliente, convertido en documento ejecutable: decisiones, fases y dónde no gastar. Prescriptivo por diseño: no inventa consejos, ordena los que ya se dieron en la reunión.

Ambas entregan HTML con estética Muta y pasan siempre por revisión humana antes de llegar al cliente.

Origen: proceso ejecutado y validado manualmente (caso "Antes de Cebar", julio 2026) antes de documentarse como skills.
